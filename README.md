# pandoc-reveal.js-slide-template
スライド資料のテンプレート ( [EzoeRyou/slide-cpp17book](https://github.com/EzoeRyou/slide-cpp17book) の構成とMakefileを元にした)

## 利用方法
- リポジトリセットアップ(git init)
- templateとしてこのリポジトリ追加(git remote add)
- マージして適用
- gpg鍵の設定を更新
- slide.mdを更新
